Data Download and Application Programmer's Interface (API)

    We provide access to NeuroElectro data in two forms:
  1. As a simple bulk download of the neurophysiological database content represented in Excel spreadsheets (~100Kb total). We expect that this will be sufficient for the majority of uses for the NeuroElectro data. Download Link (historical) Download Link (current database release)
  2. Through a RESTful API, allowing live, web-based dynamic query of the database's content. RESTful API Documentation.

If you have questions using the NeuroElectro data or accessing data from the API, please submit your question to the NeuroElectro Google Groups mailing list.

Please view the Publications Page for guidelines on how to cite data obtained from this resource.
